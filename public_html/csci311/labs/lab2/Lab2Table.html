A Table made with div and span elements, only

This table consists of a main div, with a div for each row, and then a span for each column. The first "row" and "columns" have special classes to style them differently for the header row

Classes are used for each element to style the cells to behave like a table. Google "display: table" and "display: table-row" etc. for how to achieve this
